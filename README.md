# Portafolio

Portafolio web modular en HTML, CSS y JavaScript.

## Estructura

- `index.html`: shell principal de la pagina.
- `assets/css/styles.css`: sistema de estilos y responsive.
- `assets/js/data/portfolioData.js`: contenido central editable.
- `assets/js/components/*.js`: componentes por seccion.
- `assets/js/app.js`: ensamblador de la app.
- `assets/js/main.js`: inicializacion de interacciones y animaciones.

## Como editar contenido

1. Abre `assets/js/data/portfolioData.js`.
2. Edita textos, links o listas.
3. Si quieres cambiar estructura visual de una seccion, modifica su componente en `assets/js/components/`.
